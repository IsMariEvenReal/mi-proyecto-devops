Proyecto DevOps con GitHub

Este sitio forma parte de una práctica de control de versiones.

Acerca del proyecto

En este breve proyecto, se utliza GitHub para simular un flujo de trabajo DevOps. ¡Próximamente tendremos una pipeline CI/CD!

Objetivo del proyecto

Practicar el control de versiones con GitHub y repasar lo aprendido en clases.

En este proyecto se manejan de forma breve y sencilla temas como el flujo colaborativo, iteraciones y buenas prácticas. ¡Estos temas forman parte de la actividad 3, la que se está realizando actualmente!
